# ML Mini Projects

A collection of small machine learning projects built to strengthen fundamentals in data preprocessing, modeling, and evaluation.

## Topics Covered
- Regression
- Classification
- Clustering
- Data preprocessing
- Model evaluation

## Tech Stack
Python • Pandas • NumPy • scikit-learn • Matplotlib

## Projects

### 1. Linear Regression
Predict numerical values using regression models.

### 2. Classification
Binary and multiclass classification using real datasets.

### 3. Clustering
Unsupervised learning using K-Means.

## Goal
Build strong foundations in machine learning while experimenting with different datasets and models.
